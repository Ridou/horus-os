# Bug reports

This forum is for confirmed defects only. For usage questions use the help forum. For feature ideas use #feature-requests or GitHub Discussions. For security issues do not post here, follow https://github.com/Ridou/horus-os/blob/main/SECURITY.md.

## How reports flow
- New posts start as open.
- A maintainer triages on a weekly cadence. Allow up to 7 days for a first response.
- If a report can be reproduced and is in scope, it moves to a GitHub issue and gets tagged confirmed with the issue link. GitHub is the source of truth from that point on.
- Status tags: open, investigating, confirmed, fixed, wontfix, duplicate. Only maintainers set status tags.

Filing a bug does not assign it to you. The maintainer holds the queue, and outside pull requests are not being accepted yet.

Use the pinned template, and redact all secrets and personal data.
