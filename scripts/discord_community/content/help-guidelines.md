# How to ask a good question

Before you post:
1. Search this forum and the docs at https://docs.horus-demo.com. Your question may already be answered.
2. Pick the tag that matches your question.

In your post:
- Use a specific title. "Slack adapter returns 401 on startup" beats "help".
- Paste code and logs in code blocks (triple backticks), not screenshots.
- Say what you expected and what actually happened.
- Include your Horus-OS version (run `pip show horus-os | grep Version`), your OS, and your Python version (run `python --version`).
- For anything failing, include the minimal steps to reproduce it.
- Redact API keys, tokens, vault paths, phone numbers, and IP addresses before pasting.

When your question is answered, mark the thread Solved (apply the Solved tag, or use the Mark Solution command if it is available) so the next person can find it.
