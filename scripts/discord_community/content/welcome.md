# Welcome to Horus-OS

Horus-OS is an open-source, self-hosted autonomous AI command center. This is the community server for people running it, extending it, and shaping where it goes.

## Get started
1. Read the rules below.
2. Say hi in #introductions.
3. Open "Channels and Roles" (top of the channel list) and pick your interests, notifications, and whether you can help.
4. Ask questions in the help forum, and show what you built in showcase.

## Community rules
1. Be respectful. This is a learning space. Assume good faith, correct gently, and keep it free of harassment, slurs, and personal attacks.
2. Use the right channel. Questions go in the help forum, not in general.
3. Search before you ask. Check the help forum and the docs first. Duplicate threads get pointed back to the original.
4. Never post personal or sensitive data. Redact API keys, tokens, vault paths, phone numbers, IP addresses, and anything account-bound before you paste. This applies to logs and screenshots too.
5. No spam, no advertising, and no unsolicited DMs.
6. English in the shared channels so everyone can follow and search keeps working.
7. GitHub is the source of truth for bugs and features. The support channels explain how reports flow there.

Full Code of Conduct: https://github.com/Ridou/horus-os/blob/main/CODE_OF_CONDUCT.md
