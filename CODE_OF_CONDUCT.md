# Code of Conduct

`horus-os` adopts the Contributor Covenant version 2.1 as its code of conduct.

The full text of Contributor Covenant 2.1 is available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

The project is solo-maintained and does not yet publish a dedicated conduct contact address. Until one is published, report a conduct concern privately by opening a draft GitHub Security Advisory at https://github.com/Ridou/horus-os/security/advisories/new (used here as the project's private contact channel; note in the report that it is a conduct matter, not a vulnerability). For non-sensitive matters, you can also open a GitHub Discussion.

## Enforcement

The project maintainers commit to responding to any reported violation within seven days and applying the Contributor Covenant enforcement ladder.
