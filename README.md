# horus-os

An open-source autonomous AI command center. Public-from-day-one sibling of the private `horus` daily-driver.

## Status

Bootstrap stage. No code yet. Roadmap and architecture being planned here in isolation from the private repo.

## Architectural rule

This repo is **not** a fork, snapshot, or extract of any private project. It is built up from scratch with public-only code, configuration, and documentation. Personal information of the maintainer never enters this tree.

## Local layout

This project lives at `~/Projects/horus-os/` as a sibling of `~/Projects/horus/`. The two repos are independent. No git history, branches, or remotes are shared.
