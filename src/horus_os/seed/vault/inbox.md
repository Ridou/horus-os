# Inbox

A single place to dump anything before you sort it. The rule is simple:
capture first, organize later. Empty this regularly by moving items to where
they belong.

## Unsorted

- A stray thought.
- A link to come back to.
- A task you do not want to forget.

## How to process this

When you process the inbox, for each item decide one of:

- Do it now if it takes two minutes.
- Move it to todo.md if it is a task.
- Move it to reading-list.md if it is something to read.
- Move it to project-ideas.md if it is an idea.
- Delete it if it no longer matters.

The goal is an empty inbox, not a perfect one.
