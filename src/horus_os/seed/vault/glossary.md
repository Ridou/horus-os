# Glossary

Shared definitions for terms you use often. Keeping them in one place means
the agents and you agree on what words mean. Add entries as they come up.

## Terms

### Agent

A named configuration that the runtime uses to answer a request. Each agent
has a system prompt, an optional persona file, and a set of allowed tools.

### Vault

This folder of markdown notes. The agents can read and write here, and it is
the long-lived memory you can edit by hand.

### Trace

A record of a single agent run: the prompt, the response, the tools used, and
the cost. Traces let you see how an answer was produced.

### Persona

The SOUL.md file that gives an agent its identity, principles, voice,
boundaries, and workflow.

### Your own term

- Definition: write your own as the need arises.
