<objective>
Add the agent_profiles table to the SQLite schema, bump SCHEMA_VERSION to 3, and wire
a CRUD API (load_profile, save_profile, list_profiles, delete_profile) onto the Database
class. Bootstrap a default agent in Database.init() via INSERT OR IGNORE so the profile
exists in every context - fresh installs, existing v0.1 databases, and test fixtures.

Purpose: Phase 12 is the storage-layer hard gate for Phases 13-17. Until this plan
ships, no multi-agent feature can read or write a named profile.
Output: AgentProfile dataclass, agent_profiles DDL, CRUD API, default agent bootstrap,
full test coverage of migration and CRUD paths.

    - Existing trace and note_write CRUD methods remain unaffected (all prior tests pass)

  <action>
    In src/horus_os/storage.py, add the following methods to the Database class in the
    order listed. Place them after list_note_writes and before the static helper methods.

    _row_to_profile(row: sqlite3.Row) -> AgentProfile:
      Deserialize the row. allowed_tools is stored as JSON text or NULL; parse with
      json.loads when non-null, return None when null (None = unrestricted per D-02).
      memory_scope and default_model pass through as-is (nullable strings).
      Map all eight columns to AgentProfile fields.
      Decorated with @staticmethod.

    load_profile(name: str) -> AgentProfile | None:
      SELECT by name, return _row_to_profile(row) or None.

    save_profile(profile: AgentProfile) -> None:
      Upsert using INSERT INTO agent_profiles (...) ... ON CONFLICT(name) DO UPDATE SET
      system_prompt=excluded.system_prompt, default_model=excluded.default_model,
      allowed_tools=excluded.allowed_tools, memory_scope=excluded.memory_scope,
      updated_at=excluded.updated_at.
      Note: created_at is NOT in the DO UPDATE clause, so it is preserved on conflict.
      Serialize allowed_tools to JSON text with json.dumps if non-null, else NULL.
      Set created_at and updated_at both to _now_iso() in the INSERT values. The
      conflict handler overwrites updated_at but keeps created_at from the original row.

    list_profiles() -> list[AgentProfile]:
      SELECT * FROM agent_profiles ORDER BY name ASC. Return list of _row_to_profile.

    delete_profile(name: str) -> bool:
      DELETE WHERE name = ?. Return True if conn.execute(...).rowcount > 0, else False.
