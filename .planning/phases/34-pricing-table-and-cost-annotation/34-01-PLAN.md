<objective>
Ship `pricing.json` as package data plus the `PricingTable` and `CostAnnotator` that turn token counts into USD costs. CostAnnotator subscribes to the existing `ObservationBus` singleton BEFORE the Phase 33 `SQLitePersister` so each `LLMCallEvent` is mutated in place; the persister then writes the populated `cost_usd` and `pricing_missing` columns. Unknown models persist with `pricing_missing=1, cost_usd=NULL` (NULL is honest, zero is a lie; Pitfall 5).

Purpose: Phase 33 left `cost_usd` NULL on every `llm_calls` row. Phase 34 is the bridge between token capture and dollar-denominated cost. Downstream Phases 35-37 (queries, dashboard, CLI) all read `total_cost_usd` and depend on this annotator running upstream of the persister.

Output: One new package-data JSON, two new observability modules (`pricing.py`, `cost.py`), two-line `__init__.py` re-exports, a `pricing_path` field on `Config` with env + TOML override, a one-line `subscribe` insertion in `create_app`, one `[tool.setuptools.package-data]` line in `pyproject.toml`, and five new test files covering the contract.
