## Architectural constraints from research

- stream_anthropic_async: use `async with AsyncAnthropic().messages.stream()` context manager,
  iterate `stream.text_stream` for text deltas. After text_stream is exhausted (still inside
  the `async with` block), call `await stream.get_final_message()` to inspect tool_use blocks.
- stream_gemini_async: use `client.aio.models.generate_content_stream()`, guard every chunk
  with `if chunk.text:` to skip empty heartbeat chunks. Accumulate ToolCallEvent objects during
  iteration; yield them after the text loop ends.
- system prompt in Anthropic: pass as `request["system"] = system` (top-level), NOT inside the
  messages list. Putting it in messages causes a 400 error from the Anthropic API.
- Do NOT handle tool execution inside run_agent_stream. Tool dispatch belongs in run_agent_loop.
  run_agent_stream emits ToolCallEvent as a notification only; callers observe it.
- Trace recording: run_agent_stream does not write to SQLite. Callers (Phase 15 CLI, Phase 16
  dashboard) are responsible for assembling yielded str chunks and recording one trace per call,
  not one per chunk. The design enables this -- callers join all str events for the final text.
- Lazy imports: AsyncAnthropic and google.genai must be imported inside the function body,
  matching the existing pattern in call_anthropic_async and call_gemini_async. This allows the
  package to load cleanly without SDKs installed.

   ```
   async def stream_gemini_async(
       prompt: str,
       *,
       model: str,
       system: str | None = None,
   ) -> AsyncGenerator[str | ToolCallEvent, None]:
       from google import genai

       api_key = _read_api_key()
       client = genai.Client(api_key=api_key) if api_key else genai.Client()
       request: dict = {"model": model, "contents": prompt}
       if system:
           from google.genai import types as genai_types
           request["config"] = genai_types.GenerateContentConfig(system_instruction=system)
       tool_events: list[ToolCallEvent] = []
       async for chunk in client.aio.models.generate_content_stream(**request):
           if chunk.text:
               yield chunk.text
           for candidate in getattr(chunk, "candidates", None) or []:
               content = getattr(candidate, "content", None)
               for part in getattr(content, "parts", None) or []:
                   fc = getattr(part, "function_call", None)
                   if fc is not None:
                       tool_events.append(ToolCallEvent(
                           name=getattr(fc, "name", ""),
                           input=dict(getattr(fc, "args", {}) or {}),
                       ))
       for event in tool_events:
           yield event
   ```
