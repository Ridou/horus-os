# horus-os - Phase 13 Plan 02

This plan wires the delegation tool and extends the agent runtime. Depends on Plan 01 artifacts.

## Project context

An open-source, self-hosted autonomous AI command center. Python 3.11+, FastAPI, SQLite (WAL mode), Next.js dashboard. Direct Anthropic and Gemini SDKs.

## Design constraints

- No new packages. stdlib threading + concurrent.futures only.
- run_agent_loop does NOT write traces itself; the delegation handler writes the sub-agent trace with parent_trace_id. The CLI and server layers write their own top-level traces as before.
- make_delegate_tool is a factory (closure). It captures db, master_registry, parent_trace_id, budget, and provider as Python variables. The tool JSON schema only exposes agent_name and task strings to the model.
- system_prompt must be included in EVERY turn's API request (not just the first), because Anthropic requires it on every messages.create call and Gemini requires it in GenerateContentConfig each time.

def run_agent_loop(
    prompt: str,
    *,
    registry: ToolRegistry,
    provider: str = "anthropic",
    model: str | None = None,
    max_iterations: int = 10,
    on_tool_result: Callable[[ToolResult], None] | None = None,
) -> AgentResult:
    _check_provider(provider)
    if max_iterations < 1:
        raise ValueError("max_iterations must be >= 1")
    conversation = _new_conversation(provider, model)
    tools = registry.list()
    result = conversation.send(prompt=prompt, tools=tools)
    iteration = 0
    while result.tool_uses and iteration < max_iterations:
        outcomes = execute_tool_uses(registry, result, on_log=on_tool_result)
        iteration += 1
        result = conversation.send(tool_results=outcomes, tools=tools)
    return result

def _new_conversation(provider: str, model: str | None) -> Any:
    if provider == "anthropic":
        return _anthropic.Conversation(model=model)
    return _gemini.Conversation(model=model)

<!-- From src/horus_os/tools/loop.py (current execute_tool_uses) -->

def execute_tool_uses(
    registry: ToolRegistry,
    result: AgentResult,
    *,
    on_log: Callable[[ToolResult], None] | None = None,
) -> list[ToolResult]:
    outcomes: list[ToolResult] = []
    for use in result.tool_uses:
        start = time.perf_counter()
        outcome = ToolResult(tool_use_id=use.id, name=use.name)
        try:
            outcome.output = registry.invoke(use.name, use.input)
        except BaseException as exc:
            outcome.error = f"{type(exc).__name__}: {exc}"
        outcome.latency_ms = int((time.perf_counter() - start) * 1000)
        outcomes.append(outcome)
        if on_log is not None:
            _call_logger(on_log, outcome)
    return outcomes

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-13-05 | Spoofing | make_delegate_tool: agent_name input | mitigate | load_profile returns None; handler returns safe error string on unknown name |
| T-13-06 | Tampering | sub-agent output injected into coordinator | accept | local process trust boundary; sub-agent output is treated as tool result, not trusted instruction |
| T-13-07 | Denial | parallel delegation launching excessive threads | accept | ThreadPoolExecutor is bounded by the number of delegate_to_agent calls in one model response (typically 1-5) |
