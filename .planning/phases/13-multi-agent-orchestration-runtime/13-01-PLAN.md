# horus-os - Phase 13 Plan 01

This plan extends the storage layer for multi-agent trace linkage and adds the iteration budget primitive.

## Project context

An open-source, self-hosted autonomous AI command center. Python 3.11+, FastAPI, SQLite (WAL mode), Next.js dashboard. Direct Anthropic and Gemini SDKs.

## Current SCHEMA_VERSION and init() pattern

`SCHEMA_VERSION = 3` in storage.py. The `init()` method:
1. Runs `executescript(SCHEMA_SQL)` - idempotent DDL (CREATE IF NOT EXISTS)
2. Reads schema_version row; inserts if missing, updates if lower than SCHEMA_VERSION
3. Bootstraps default agent profile via `INSERT OR IGNORE`

For v3->v4: SCHEMA_SQL must add the new columns to the `traces` CREATE TABLE (for fresh databases). The migration block in init() must ALTER TABLE for existing v3 databases using try/except OperationalError (ALTER TABLE is not idempotent).
