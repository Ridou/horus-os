<!-- Backend contract from Plan 73-02 (the endpoints this page consumes). -->

Backend endpoints (Plan 73-02):
- POST /api/research { question } -> { task_id, plan: { subtopics: [{ query, goal }] } }  (plan-only, no execution)
- POST /api/research/{id}/start -> 200 (flips to running, schedules background run)
- GET  /api/research/{id}/progress -> { phase: "plan"|"searching"|"reading"|"synthesizing"|"done"|"cancelled", sources_found: number, iterations_used: number, iteration_budget: number }
- GET  /api/research/{id}/report -> rendered cited markdown (404/409 until completed)
- POST /api/research/{id}/cancel -> 200 (status cancelled)
