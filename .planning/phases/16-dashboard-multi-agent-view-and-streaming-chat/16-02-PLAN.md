## SSE consumption in the browser (no library)

```
const r = await fetch("/api/chat/stream", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ prompt }),
});
const reader = r.body.getReader();
const decoder = new TextDecoder();
let buffer = "";
while (true) {
  const { value, done } = await reader.read();
  if (done) break;
  buffer += decoder.decode(value, { stream: true });
  // SSE frames are separated by "\n\n"
  const parts = buffer.split("\n\n");
  buffer = parts.pop() || "";
  for (const part of parts) {
    if (!part.startsWith("data: ")) continue;
    const payload = JSON.parse(part.slice(6));
    dispatch(payload);
  }
}
```

`dispatch(payload)` reads `payload.type` and appends text, surfaces a
tool-call line, finalizes the trailing meta line, or paints an error.

## Delegate tree UX

- Each row in the Traces table gets a small `expand` button when the
  row has `agent_profile_name != null` OR is a parent (we cannot know
  from the list endpoint whether a row has children without a JOIN;
  v0.2 displays the expand button when the row has an agent name,
  trusting that the user will click and the child loader will either
  populate or render `(no children)`).
- On click, the row toggles to a two-row layout: the original row,
  plus a nested `<tr>` containing a `<table>` of child traces. The
  child table reuses the same columns (created_at, provider, model,
  status, prompt) plus an `agent_profile_name` column.
- Two levels deep; child rows do not themselves get expand buttons in
  v0.2. The CONTEXT records the v0.3 deferred follow-up.

## v0.1 compatibility hard rule

- The chat surface always falls back gracefully: if the SSE stream
  emits no `token` frames before a `done`, the user sees the meta
  line only. This matches today's "(no text response)" placeholder.
- The trace table renders parent_trace_id and agent_profile_name when
  present and shows nothing when null. The `expand` affordance is
  skipped when `agent_profile_name` is null.

```
$("#send").addEventListener("click", async () => {
  const prompt = $("#prompt").value.trim();
  if (!prompt) return;
  $("#send").disabled = true;
  $("#chat-status").textContent = "running ...";
  const responseRoot = $("#chat-response");
  responseRoot.innerHTML = "";
  const pre = document.createElement("pre");
  responseRoot.appendChild(pre);
  const status = document.createElement("div");
  status.className = "status";
  responseRoot.appendChild(status);
  const start = performance.now();
  try {
    const r = await fetch("/api/chat/stream", {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({ prompt }),
    });
    if (!r.ok) {
      const body = await r.json().catch(() => ({}));
      const detail = typeof body.detail === "string" ? body.detail : JSON.stringify(body.detail || body);
      pre.classList.add("stream-error");
      pre.textContent = `${r.status} ${detail}`;
      $("#chat-status").textContent = "";
      $("#send").disabled = false;
      return;
    }
    const reader = r.body.getReader();
    const decoder = new TextDecoder();
    let buffer = "";
    let traceId = null;
    let latencyMs = null;
    let provider = "?";
    let model = "?";
    let toolCount = 0;
    let sawError = false;
    while (true) {
      const { value, done } = await reader.read();
      if (done) break;
      buffer += decoder.decode(value, { stream: true });
      const parts = buffer.split("\n\n");
      buffer = parts.pop() || "";
      for (const part of parts) {
        const line = part.trim();
        if (!line.startsWith("data: ")) continue;
        let payload;
        try { payload = JSON.parse(line.slice(6)); } catch (e) { continue; }
        if (payload.type === "token") {
          pre.appendChild(document.createTextNode(payload.text));
        } else if (payload.type === "tool_call") {
          toolCount += 1;
          const aside = document.createElement("div");
          aside.className = "stream-tool";
          aside.textContent = `[tool-request] ${payload.name}(${JSON.stringify(payload.input)})`;
          responseRoot.insertBefore(aside, status);
        } else if (payload.type === "done") {
          traceId = payload.trace_id;
          latencyMs = payload.latency_ms;
          provider = payload.provider || provider;
          model = payload.model || model;
        } else if (payload.type === "error") {
          sawError = true;
          pre.classList.add("stream-error");
          pre.appendChild(document.createTextNode(`\n[error] ${payload.message}`));
        }
      }
    }
    if (latencyMs === null) latencyMs = Math.round(performance.now() - start);
    const meta = sawError
      ? `[error after ${latencyMs}ms${traceId ? ", trace " + traceId.slice(0, 8) : ""}]`
      : `[streamed, ${latencyMs}ms${traceId ? ", trace " + traceId.slice(0, 8) : ""}, ${toolCount} tool calls]`;
    status.textContent = meta;
    $("#chat-status").textContent = "";
  } catch (e) {
    pre.classList.add("stream-error");
    pre.textContent = e.message || String(e);
    $("#chat-status").textContent = "";
  } finally {
    $("#send").disabled = false;
  }
});
```

fallback `?` values; the meta line is informational. Do not add

```
async function loadAgents() {
  const body = $("#agents-body");
  body.innerHTML = '<div class="empty">loading</div>';
  try {
    const data = await fetchJson("/api/agents");
    if (!data.agents.length) {
      body.innerHTML = '<div class="empty">no agent profiles yet</div>';
      return;
    }
    const rows = data.agents.map((a) => {
      const model = a.default_model || "(default)";
      const tools = a.allowed_tools === null ? "(all)" : (a.allowed_tools.join(", ") || "(none)");
      const last = a.last_activity_at || "(never)";
      const prompt = (a.system_prompt || "").length > 60
        ? a.system_prompt.slice(0, 60) + "..."
        : (a.system_prompt || "");
      return `<tr><td>${escapeHtml(a.name)}</td><td>${escapeHtml(model)}</td><td>${escapeHtml(tools)}</td><td class="muted">${escapeHtml(last)}</td><td class="muted">${escapeHtml(prompt)}</td></tr>`;
    }).join("");
    body.innerHTML = `<table><thead><tr><th>name</th><th>default_model</th><th>allowed_tools</th><th>last_activity_at</th><th>system_prompt</th></tr></thead><tbody>${rows}</tbody></table>`;
  } catch (e) {
    body.innerHTML = `<div class="empty">${escapeHtml(e.message)}</div>`;
  }
}
$("#refresh-agents").addEventListener("click", loadAgents);
```

```
async function loadTraces() {
  const body = $("#traces-body");
  body.innerHTML = '<div class="empty">loading</div>';
  try {
    const data = await fetchJson("/api/traces?limit=50");
    if (!data.traces.length) {
      body.innerHTML = '<div class="empty">no traces yet</div>';
      return;
    }
    const rows = data.traces.map((t) => {
      const status = t.status === "success"
        ? '<span class="pill success">success</span>'
        : '<span class="pill error">error</span>';
      const prompt = t.prompt.length > 60 ? t.prompt.slice(0, 60) + "..." : t.prompt;
      const agent = t.agent_profile_name || "";
      const expand = t.agent_profile_name
        ? `<button class="tree-toggle" data-trace="${escapeHtml(t.trace_id)}">expand</button>`
        : "";
      return `<tr data-trace-row="${escapeHtml(t.trace_id)}"><td class="muted">${escapeHtml(t.created_at)}</td><td>${escapeHtml(t.provider)}</td><td>${escapeHtml(t.model)}</td><td>${status}</td><td>${escapeHtml(agent)}</td><td>${escapeHtml(prompt)}</td><td>${expand}</td></tr>`;
    }).join("");
    body.innerHTML = `<table><thead><tr><th>created_at</th><th>provider</th><th>model</th><th>status</th><th>agent</th><th>prompt</th><th></th></tr></thead><tbody>${rows}</tbody></table>`;
    body.querySelectorAll(".tree-toggle").forEach((btn) => {
      btn.addEventListener("click", () => toggleChildren(btn));
    });
  } catch (e) {
    body.innerHTML = `<div class="empty">${escapeHtml(e.message)}</div>`;
  }
}

async function toggleChildren(btn) {
  const traceId = btn.dataset.trace;
  const row = btn.closest("tr");
  const next = row.nextElementSibling;
  if (next && next.classList.contains("tree-row")) {
    next.remove();
    btn.textContent = "expand";
    return;
  }
  btn.textContent = "loading ...";
  try {
    const data = await fetchJson(`/api/traces/${encodeURIComponent(traceId)}/children`);
    const childRows = data.children.map((c) => {
      const status = c.status === "success"
        ? '<span class="pill success">success</span>'
        : '<span class="pill error">error</span>';
      const prompt = c.prompt.length > 60 ? c.prompt.slice(0, 60) + "..." : c.prompt;
      return `<tr><td class="muted">${escapeHtml(c.created_at)}</td><td>${escapeHtml(c.provider)}</td><td>${escapeHtml(c.model)}</td><td>${status}</td><td>${escapeHtml(c.agent_profile_name || "")}</td><td>${escapeHtml(prompt)}</td></tr>`;
    }).join("");
    const inner = data.children.length
      ? `<table><thead><tr><th>created_at</th><th>provider</th><th>model</th><th>status</th><th>agent</th><th>prompt</th></tr></thead><tbody>${childRows}</tbody></table>`
      : '<div class="empty">no children</div>';
    const tr = document.createElement("tr");
    tr.className = "tree-row";
    const td = document.createElement("td");
    td.colSpan = 7;
    td.innerHTML = inner;
    tr.appendChild(td);
    row.parentNode.insertBefore(tr, row.nextSibling);
    btn.textContent = "collapse";
  } catch (e) {
    btn.textContent = "expand";
    alert(e.message);
  }
}
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| SSE frame -> DOM | JSON payload bodies (token text, tool input, error message) are written into DOM nodes |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-16-08 | Tampering | Token text written to DOM | mitigate | `appendChild(document.createTextNode(...))` for streamed tokens; tool aside uses `.textContent =`; trace renderer routes every string through `escapeHtml` before innerHTML |
| T-16-09 | XSS | Trace `prompt` or `agent_profile_name` containing HTML | mitigate | escapeHtml() wraps every interpolated string in the templated rows |
| T-16-10 | Information Disclosure | Browser network tab shows the SSE stream | accept | Local-only dashboard; user owns the network |
| T-16-11 | Denial | Long stream holds the browser tab busy | accept | Same constraint as v0.1 chat; the response area updates live so the user can cancel by navigating |
</threat_model>
