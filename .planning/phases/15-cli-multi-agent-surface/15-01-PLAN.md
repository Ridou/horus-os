# horus-os - Phase 15 Plan 01

This plan adds the `agents` CRUD subcommand. The Phase 12 storage API
(`load_profile`, `save_profile`, `list_profiles`, `delete_profile`) is already
in place and tested; this plan is pure CLI layer.

## Established subcommand pattern (from cli/traces_cmd.py and cli/run_cmd.py)

Every subcommand handler is sync (no async), receives args plus injected
stdout/stderr TextIO, returns `int` exit code, and lives in its own
`cli/<name>_cmd.py` module. The handler short-circuits with exit 1 and an error
message on `stderr` when the database is missing, mirroring traces_cmd. Tests
use `io.StringIO()` buffers passed through `main(argv, stdout=..., stderr=...)`.

## AgentProfile shape (from types.py, AgentProfile)

```
name: str
system_prompt: str
default_model: str | None = None
allowed_tools: list[str] | None = None  # None means unrestricted
memory_scope: str | None = None
created_at: str = ""  # set by Database
updated_at: str = ""  # set by Database
```

## Database methods (from storage.py, verified)

```
db.list_profiles() -> list[AgentProfile]
db.load_profile(name: str) -> AgentProfile | None
db.save_profile(profile: AgentProfile) -> None   # upsert; preserves created_at on conflict
db.delete_profile(name: str) -> bool              # True if a row was deleted
```

The `init()` call seeds a `default` profile, so a freshly initialized
installation always has at least one row to list.

## Nested argparse pattern (research §Pattern 1)

```
agents_p = sub.add_parser("agents", help="Manage agent profiles")
agents_sub = agents_p.add_subparsers(dest="agents_command", metavar="<operation>")
agents_p.set_defaults(func=run_agents)
list_p = agents_sub.add_parser("list", ...)
show_p = agents_sub.add_parser("show", ...)
create_p = agents_sub.add_parser("create", ...)
edit_p = agents_sub.add_parser("edit", ...)
delete_p = agents_sub.add_parser("delete", ...)
```

`run_agents` dispatches on `args.agents_command`. When unspecified (the user
typed `horus-os agents` bare), treat it as `list` for a friendly default.

## allowed_tools on the CLI

`AgentProfile.allowed_tools` is a list of tool names or None (unrestricted).
For `create` and `edit`, accept a comma-separated string via `--allowed-tools`
and convert to a list. The sentinel string "all" or omitted flag means
`allowed_tools = None`. An empty list explicitly disables tools.

```
"""`horus-os agents` subcommand."""

from __future__ import annotations

import argparse
from typing import TextIO

from horus_os.config import Config
from horus_os.storage import Database
from horus_os.types import AgentProfile


def run_agents(args: argparse.Namespace, *, stdout: TextIO, stderr: TextIO) -> int:
    config = Config.load(getattr(args, "data_dir", None))
    if not config.db_path.exists():
        stderr.write(f"No database at {config.db_path}. Run `horus-os init` first.\n")
        return 1
    db = Database(config.db_path)
    op = getattr(args, "agents_command", None) or "list"
    if op == "list":
        return _cmd_list(db, stdout)
    if op == "show":
        return _cmd_show(db, args.name, stdout, stderr)
    if op == "create":
        return _cmd_create(db, args, stdout, stderr)
    if op == "edit":
        return _cmd_edit(db, args, stdout, stderr)
    if op == "delete":
        return _cmd_delete(db, args.name, stdout, stderr)
    stderr.write(f"Unknown agents operation: {op!r}\n")
    return 2
```

   - profile = db.load_profile(args.name)
   - If None: stderr.write(f"No agent profile named {args.name!r}.\n"); return 1
   - If args.system_prompt is not None: profile.system_prompt = args.system_prompt
   - If args.model is not None: profile.default_model = args.model
   - If args.allowed_tools is not None: profile.allowed_tools = _parse_allowed_tools(args.allowed_tools)
   - If args.memory_scope is not None: profile.memory_scope = args.memory_scope
   - db.save_profile(profile)
   - stdout.write(f"Updated agent profile {args.name!r}.\n"); return 0

7. Add `_cmd_delete(db, name, stdout, stderr)`:
   - deleted = db.delete_profile(name)
   - If not deleted: stderr.write(f"No agent profile named {name!r}.\n"); return 1

   - If raw.strip() == "": return []

| T-15-04 | Spoofing | delete arbitrary profile name | accept | local install; user owns the database |
