sqlite-vec (verified PyPI 0.1.9, all three OS wheels):
- `import sqlite_vec` then `conn.enable_load_extension(True); sqlite_vec.load(conn); conn.enable_load_extension(False)`.
- `from sqlite_vec import serialize_float32` converts a `list[float]` to the bytes blob.
- vec0 virtual table with a text primary key plus a float column:
  `CREATE VIRTUAL TABLE IF NOT EXISTS note_vectors USING vec0(rel_path TEXT PRIMARY KEY, embedding FLOAT[<dim>])`
- Insert: `conn.execute("INSERT OR REPLACE INTO note_vectors(rel_path, embedding) VALUES (?, ?)", (rel_path, serialize_float32(vec)))`.
- KNN: `SELECT rel_path, distance FROM note_vectors WHERE embedding MATCH ? ORDER BY distance LIMIT ?` with `serialize_float32(query_vec)` as the bound param.
- NOTE: Python's bundled sqlite3 must have `enable_load_extension`. Guard the load with a try/except and surface a clear message (STACK.md notes pyenv-compiled Python may disable it).

EmbeddingBackend (from Plan 70-01, src/horus_os/memory/embeddings.py):
- `ONNXEmbeddingBackend(model_name, models_dir)`; `.dimension: int`; `.model_name: str`; `.is_model_present() -> bool`; `.embed(texts) -> list[list[float]]` (raises `EmbeddingModelMissingError` when absent).

NotesStore (src/horus_os/memory/notes.py): `__init__(self, notes_dir, *, on_write=None)`; `create_note`/`append_note` build a NoteWrite, call `self._notify(write)` (the audit callback), and return it; `list_notes()`/`read_note(rel)` exist for reindex walking; `_resolve` enforces path safety.

Config (src/horus_os/config.py, extended in 70-01): `vector_memory_enabled`, `embedding_model`, `models_path()`. Add a `vectors_path()` helper here returning `self.data_dir / "vectors.sqlite"`.
